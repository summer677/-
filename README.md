# 🍅 番茄时钟 (Tomato Clock)

一款基于番茄工作法的专注力管理工具，帮助你高效管理时间、追踪任务和应用使用情况。

## ✨ 功能特性

- 🎯 **专注计时** — 25分钟专注 + 5分钟短休 + 15分钟长休，科学的时间管理
- ✅ **任务管理** — 添加/完成/删除任务，支持分类（学习/工作/生活）和截止日期
- 🛏️ **起床/睡眠打卡** — 记录每日作息，养成规律习惯
- 📱 **应用使用统计** — 记录各应用使用时长，直观查看时间分配
- 📊 **周报总览** — 本周起床/睡眠趋势、应用使用时长 TOP5、累计专注次数
- 📈 **数据统计** — 连续专注天数、任务完成率、累计专注分钟数
- 🌍 **多语言支持** — 中文 / English / 日本語 / 한국어

## 🛠️ 技术栈

- 前端：纯 HTML + CSS + JavaScript（无框架依赖）
- 后端：Supabase（PostgreSQL + 身份认证 + Row Level Security）
- 部署：Vercel
